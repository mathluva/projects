Global Suicide Trends

Tools: Python 3, Matplotlib, Seaborn, Pandas, NumPy, SciPy, GitHub

Worked with a team to import and clean global suicide data for the years 1985 to 2016. The suicide data was categorized by country, year, gender, age, number of suicides, population, generation, and growth domestic product (gdp) per capita for each country. We analyzed global trends, suicides and gender, suicides and age groups and suicide trends in the USA. In addition, we used scipy.stats to perform an independent t-test to determine if their was any statistical significant difference in the average number of suicides in males versus females.

Python API Project

Tools: HTML, CSS, Bootstrap, Python 3, APIs, Matplotlib, Pandas, Jupyter Notebook

Created a Python script to pull data from the OpenWeatherMap API to assemble a dataset on over 500 cities. Performed a weather check on each of the cities using a series of successive API calls. Included a print log of each city as it was being processed with the city number and city name. After assembling the dataset, I used Matplotlib to plot various aspects of the weather versus latitude and created a series of scatter plots to showcase the relationships.
